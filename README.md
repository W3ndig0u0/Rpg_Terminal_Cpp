# RPG Terminal C++ ⚔️

A **turn-based RPG** engine designed for the command line. This project showcases the use of **Object-Oriented Programming (OOP)** in C++ to create a structured and expandable game world.

## ✨ Features
*   **Class System**: Choose from distinct hero classes, each with unique stats and abilities.
*   **Turn-Based Combat**: Strategic battle system against various monster types.
*   **Dynamic Exploration**: Navigate through different areas like villages and dungeons via terminal commands.
*   **XP & Leveling**: Defeat enemies to gain experience points and strengthen your hero.
*   **Colored UI**: Enhanced terminal visuals using ANSI escape codes for better readability.

## 🛠️ Tech Stack
*   **Language**: C++ (utilizing templates and OOP principles).
*   **Input/Output**: Standard `iostream` with `termios.h` (for specialized terminal handling).
*   **Build System**: Makefile included for easy compilation.
